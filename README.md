# HabitPulse — Smart Habit Reminder System (Visual Prototype)

This project is a **high-fidelity SaaS landing page prototype** for HabitPulse.

## What this is
- A visually rich, animated marketing website.
- Built with plain HTML, CSS, and JavaScript.
- Focused entirely on storytelling, UI polish, and motion design.

## What this is not
- Not a functional reminder app.
- No backend, authentication, or real form submission.

## Included sections
1. Sticky glass navigation + mobile fullscreen menu
2. Hero with animated badge, gradient headline, CTAs, trust row
3. Infinite logo ticker
4. Bento feature grid
5. 3-step how-it-works with animated connector
6. App preview mockup with floating 3D feel
7. Testimonials grid
8. Count-up statistics section
9. Pricing cards with monthly/yearly toggle
10. Final gradient CTA
11. Multi-column footer

## Motion and interactions
- Page loader animation
- Scroll progress bar
- IntersectionObserver-based reveal animations
- Hover micro-interactions on cards/buttons/links
- Animated back-to-top button
- Optional desktop custom cursor

## Run
Open `index.html` directly, or use a static server:

```bash
python3 -m http.server 8000
```
